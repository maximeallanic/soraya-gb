# Soraya GB

A Game Boy homebrew RPG starring **Soraya** the cat.

Built with [GB Studio](https://www.gbstudio.dev/) 4.x.

## Story

Soraya, a brave orange cat, embarks on an adventure through the bureau and beyond...

## Assets

| File | Description |
|------|-------------|
| `assets/sprites/soraya.png` | Soraya player sprite — 16×16 RGBA, 5-color palette |
| `assets/backgrounds/bureau.png` | Bureau starting scene — 160×144, GB 4-color palette |

## Development

### Prerequisites

- [GB Studio](https://www.gbstudio.dev/) 4.x

### Open project

Open `project.gbsproj` in GB Studio.
